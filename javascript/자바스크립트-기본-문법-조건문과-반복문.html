자바스크립트 기본 문법: 조건문과 반복문 쉽게 알아보기

자바스크립트에서 조건문과 반복문은 프로그램의 흐름을 제어하고 반복 작업을 수행하는데 사용됩니다. 이번 글에서는 조건문과 반복문의 기본적인 사용법에 대해 살펴봅시다.

if 문

if문은 주어진 조건이 참인지 거짓인지에 따라 코드 블록을 실행합니다.

javascript
let x = 10;

if (x > 5) {
  console.log("x는 5보다 큽니다.");
} else {
  console.log("x는 5보다 작거나 같습니다.");
}

switch 문

switch문은 여러 경우 중 하나에 따라 코드 블록을 실행합니다.

javascript
let day = 3;
let dayName;

switch (day) {
  case 1:
    dayName = "월요일";
    break;
  case 2:
    dayName = "화요일";
    break;
  default:
    dayName = "기타 요일";
    break;
}
console.log(dayName);

for 문

for문은 주어진 조건에 따라 코드 블록을 반복 실행합니다.

javascript
for (let i = 0; i < 5; i++) {
  console.log(i);
}

while 문

while문은 조건이 참인 경우 코드 블록을 반복 실행합니다.

javascript
let count = 0;

while (count < 3) {
  console.log(count);
  count++;
}

do...while 문

do...while문은 조건을 먼저 확인하지 않고 먼저 한 번 실행한 후 조건을 확인합니다.

javascript
let num = 0;

do {
  console.log(num);
  num++;
} while (num < 3);

조건문과 반복문은 자바스크립트 프로그래밍에서 중요한 개념이니 잘 숙지하고 활용해보세요! 🌟
