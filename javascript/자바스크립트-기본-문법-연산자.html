자바스크립트 기본 문법: 연산자를 쉽게 이해하기

자바스크립트에서는 다양한 연산자를 사용하여 변수나 값들을 계산하고 비교할 수 있습니다. 연산자를 제대로 이해하면 프로그램을 더욱 효율적으로 작성할 수 있습니다. 이번 글에서는 자바스크립트의 기본 연산자에 대해 알아보겠습니다.

산술 연산자

산술 연산자는 숫자 값을 다룰 때 사용됩니다.

  • + : 덧셈
  • - : 뺄셈
  • * : 곱셈
  • / : 나눗셈
  • % : 나머지

예를 들어, let result = 10 + 5;는 10과 5를 더한 값을 result 변수에 저장합니다.

비교 연산자

비교 연산자는 두 값을 비교하여 논리적인 참(true) 또는 거짓(false)을 반환합니다.

  • > : 초과
  • < : 미만
  • >= : 이상
  • <= : 이하
  • === : 일치
  • !== : 불일치

예를 들어, console.log(10 > 5);는 true를 반환합니다.

논리 연산자

논리 연산자는 논리 연산에 사용되며, 두 개 이상의 논리적인 표현을 결합하여 새로운 결과를 반환합니다.

  • && : AND
  • || : OR
  • ! : NOT

예를 들어, let x = true; let y = false; console.log(x && y);는 false를 반환합니다.

연산자 우선순위

연산자에는 우선순위가 있습니다. 괄호를 사용하여 연산의 우선순위를 조절할 수 있습니다.

예를 들어, let result = 10 + 5 * 2;에서 * 연산이 먼저 수행됩니다. 이를 피하기 위해 괄호를 사용하여 연산 우선순위를 명확히 할 수 있습니다.
