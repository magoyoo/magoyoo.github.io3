SwiftUI의 버튼과 액션 처리 방법

SwiftUI는 애플의 Swift 프로그래밍 언어를 사용하여 사용자 인터페이스를 쉽게 구성할 수 있게 해주는 프레임워크입니다. 이 글에서는 SwiftUI로 버튼을 생성하는 방법과 버튼을 눌렀을 때 액션을 처리하는 방법에 대해 알아보겠습니다.

버튼 생성하기

SwiftUI에서 버튼을 만드는 것은 매우 간단합니다. Button 뷰를 사용하여 버튼을 생성할 수 있는데, Button은 여러 가지 초기화 메서드를 제공하여 다양한 스타일과 액션을 지정할 수 있습니다. 가장 기본적인 형태는 다음과 같습니다.

swift
import SwiftUI

struct ContentView: View {
    var body: some View {
        Button("Click Me") {
            print("Button was clicked!")
        }
    }
}
