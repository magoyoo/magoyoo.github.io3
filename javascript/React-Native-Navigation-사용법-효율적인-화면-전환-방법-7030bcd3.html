React Native Navigation 사용법: 효율적인 화면 전환 방법

React Native는 크로스 플랫폼 모바일 앱 개발을 가능하게 하는 강력한 프레임워크로, JavaScript와 React를 사용하여 iOS와 Android에서 실행 가능한 네이티브 애플리케이션을 만들 수 있습니다. 그러나 앱이 복잡해지면 화면 전환과 상태 관리를 효율적으로 처리하는 것이 중요해지기 마련입니다. 이때 유용한 도구가 React Native Navigation입니다. 이 글에서는 React Native Navigation을 사용하여 효율적인 화면 전환을 구현하는 방법을 단계별로 살펴보겠습니다.

React Native Navigation 소개

React Native Navigation은 기본적으로 react-navigation 패키지를 사용하여 내비게이션을 관리합니다. 이 패키지는 단일 화면에서 탭과 스택을 구성할 수 있으며, 웹 페이지처럼 각 화면을 이동할 수 있게 도와줍니다. 가장 큰 장점은 다중 화면 및 복잡한 내비게이션 구조를 쉽게 관리할 수 있다는 점입니다.

설치와 기본 설정부터 시작하여, 화면 간의 기본 전환 방법 및 복잡한 네비게이션 구조와 상태 관리를 다루는 방법을 통해 완료된 모바일 앱을 개발하는 데 필요한 기술을 익히게 됩니다.

React Native Navigation 설치

React Native 프로젝트에서 React Navigation을 사용하려면 다음 명령어를 실행하여 관련 패키지를 설치합니다:

bash
npm install @react-navigation/native
expo install react-native-screens react-native-safe-area-context

이후, 필요한 네비게이터 패키지를 추가로 설치해야 합니다. 예를 들어, 스택 네비게이션을 사용하려면 다음 명령어를 실행합니다:

bash
npm install @react-navigation/stack

이제 프로젝트에 React Native Navigation 관련 종속성이 모두 설치되었습니다. 다음은 네비게이터를 설정하는 방법입니다.

기본 내비게이션 설정

첫 번째로, navigation 폴더를 생성하고, 그 내부에 AppNavigator.js 파일을 생성합니다. 이 파일 내에서는 네비게이터를 정의하고 필요한 설정을 합니다.

javascript
import * as React from 'react';
import { NavigationContainer } from '@react-navigation/native';
import { createStackNavigator } from '@react-navigation/stack';

import HomeScreen from '../screens/HomeScreen';
import DetailsScreen from '../screens/DetailsScreen';

const Stack = createStackNavigator();

function AppNavigator() {
  return (
    <NavigationContainer>
      <Stack.Navigator initialRouteName="Home">
        <Stack.Screen name="Home" component={HomeScreen} />
        <Stack.Screen name="Details" component={DetailsScreen} />
      </Stack.Navigator>
    </NavigationContainer>
  );
}

export default AppNavigator;

이 코드는 두 개의 스크린 HomeScreen과 DetailsScreen을 포함한 스택 네비게이터를 정의합니다. initialRouteName을 Home으로 설정하여 앱이 시작될 때 기본 화면으로 HomeScreen을 표시합니다.

화면 전환 구현 방법

이제 화면 전환을 구현해 보겠습니다. HomeScreen에서 버튼을 클릭하여 DetailsScreen으로 이동할 수 있도록 하겠습니다. HomeScreen 컴포넌트는 다음과 같습니다:

javascript
import * as React from 'react';
import { Button, View, Text } from 'react-native';

function HomeScreen({ navigation }) {
  return (
    <View style={{ flex: 1, justifyContent: 'center', alignItems: 'center' }}>
      <Text>Home Screen</Text>
      <Button
        title="Go to Details"
        onPress={() => navigation.navigate('Details')} />
    </View>
  );
}

export default HomeScreen;

이 코드는 navigation 객체를 사용하는 버튼을 포함하고 있어 Details 화면으로 이동합니다. DetailsScreen 컴포넌트는 다음과 같습니다:

javascript
import * as React from 'react';
import { View, Text } from 'react-native';

function DetailsScreen() {
  return (
    <View style={{ flex: 1, justifyContent: 'center', alignItems: 'center' }}>
      <Text>Details Screen</Text>
    </View>
  );
}

export default DetailsScreen;

이제 버튼을 클릭하면 HomeScreen에서 DetailsScreen으로 전환될 것입니다.

상태 관리와 데이터 전달

화면 전환 시 데이터를 전달해야 하는 경우도 자주 발생합니다. navigate 함수의 두 번째 인자로 데이터를 전달할 수 있습니다. 예를 들어, HomeScreen에서 다음과 같이 데이터를 전달할 수 있습니다:

javascript
<Button
  title="Go to Details"
  onPress={() => navigation.navigate('Details', { itemId: 86, otherParam: 'anything you want here', })} />

DetailsScreen에서 전달된 데이터를 액세스하려면 route 객체를 사용합니다:

javascript
function DetailsScreen({ route }) {
  const { itemId, otherParam } = route.params;
  return (
    <View style={{ flex: 1, justifyContent: 'center', alignItems: 'center' }}>
      <Text>Details Screen</Text>
      <Text>itemId: {itemId}</Text>
      <Text>otherParam: {otherParam}</Text>
    </View>
  );
}

route.params를 통해 전달된 값을 추출하고 화면에 표시할 수 있습니다.

네비게이션 간소화

다중 화면 전환 및 복잡한 네비게이션 구조를 효율적으로 관리하기 위해, react-navigation에는 다양한 네비게이터가 제공됩니다. 예를 들어, 탭 내비게이션을 사용하여 하단 탭 바를 추가하거나 드로어 네비게이션을 사용하여 측면 메뉴를 구현할 수 있습니다.

결론

React Native Navigation은 직관적인 API와 다양한 커스터마이징 옵션을 제공하여 모바일 애플리케이션 개발 시 화면 전환을 효율적으로 관리할 수 있게 해줍니다. 이 글에서는 기본적인 설치와 설정, 화면 전환의 구현 방법에 대해 살펴보았습니다. 추가적인 네비게이션 옵션과 고급 기능들을 고려하여 더 복잡한 애플리케이션을 개발해보세요.
