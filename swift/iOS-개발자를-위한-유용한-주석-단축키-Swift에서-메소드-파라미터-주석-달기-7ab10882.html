iOS 개발은 코드의 가독성과 유지보수성을 높이기 위해 다양한 방법을 사용합니다. 그 중 주석은 코드의 의미를 설명하고 협업을 원활히 하는 데 큰 역할을 합니다. 이번 글에서는 iOS 개발자가 효율적으로 주석을 달 수 있는 단축키와 Swift에서 메소드 파라미터에 주석을 다는 방법에 대해 알아보겠습니다.

기본 주석 단축키

주석은 소스 코드에 첨가되어 코드를 설명하거나 메모를 추가하는 데 사용됩니다. iOS 개발에서 자주 사용되는 주석 단축키는 다음과 같습니다:

한 줄 주석

한 줄 주석은 //를 사용하여 작성할 수 있습니다. Xcode에서는 Command + /를 사용하여 선택된 줄에 한 줄 주석을 쉽게 추가할 수 있습니다.

swift
// 이 주석은 한 줄 주석입니다.

여러 줄 주석

여러 줄 주석은 /* */를 사용하여 작성할 수 있습니다. Xcode에서는 여러 줄을 선택한 후 Command + Option + /를 사용하여 여러 줄 주석을 쉽게 추가할 수 있습니다.

swift
/*
 이 주석은
 여러 줄 주석입니다.
 */

Swift에서 메소드 파라미터 주석 달기

Swift에서는 메소드나 함수의 파라미터에 주석을 달아 코드의 가독성을 높일 수 있습니다. 특히 외부 파라미터 이름과 내부 파라미터 이름을 분리하여 명확하게 주석을 다는 것이 중요합니다.

외부 파라미터 이름과 내부 파라미터 이름

Swift 메소드에서는 외부 파라미터 이름과 내부 파라미터 이름을 명확하게 구분하여 사용할 수 있습니다. 외부 파라미터 이름은 메소드를 호출할 때 사용되고, 내부 파라미터 이름은 메소드 내에서 사용됩니다.

swift

    print("Hello, \(name)!")
