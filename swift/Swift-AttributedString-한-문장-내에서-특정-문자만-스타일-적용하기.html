Swift AttributedString 한 문장 내에서 특정 문자만 스타일 적용하기!

Swift는 강력한 텍스트 스타일링 기능을 제공하여 개발자들이 더욱 매력적이고 사용하기 쉬운 사용자 인터페이스를 만들 수 있도록 돕습니다. 특히, Swift의 AttributedString을 사용하면 한 문장 내에서 특정 문자나 단어에 다양한 스타일을 적용할 수 있습니다. 이 글에서는 AttributedString을 사용하여 텍스트의 특정 부분에 스타일을 적용하는 방법을 상세히 살펴보겠습니다.

AttributedString이란?

AttributedString은 텍스트에 다양한 속성을 적용할 수 있는 문자열 타입입니다. 이를 통해 문자열의 특정 부분에 글꼴, 색상, 굵기, 기울임 등 다양한 스타일을 개별적으로 적용할 수 있습니다. 이는 텍스트의 가독성을 높이고, 사용자가 중요한 정보를 쉽게 식별할 수 있도록 도와줍니다.

AttributedString의 기본 사용법

먼저, AttributedString의 기본 사용법을 간단한 예제를 통해 살펴보겠습니다.

swift
import SwiftUI

struct ContentView: View {
    var body: some View {
        let fullString = "SwiftUI는 정말 강력한 프레임워크입니다."
        var attributedString = AttributedString(fullString)

        if let range = attributedString.range(of: "SwiftUI") {
            attributedString[range].font = .system(size: 24, weight: .bold)
        }

        if let range = attributedString.range(of: "강력한") {
            attributedString[range].foregroundColor = .red
        }

        return Text(attributedString)
            .padding()
    }
}

struct ContentView_Previews: PreviewProvider {
    static var previews: some View {
        ContentView()
    }
}

이 예제에서는 "SwiftUI" 텍스트를 굵고 큰 글씨로, "강력한" 텍스트를 빨간색으로 표시했습니다. 이를 통해 AttributedString을 사용하여 텍스트의 특정 부분에 스타일을 적용하는 방법을 간단히 확인할 수 있습니다.

다양한 스타일 적용하기

AttributedString을 사용하면 텍스트의 특정 부분에 대해 다양한 스타일을 적용할 수 있습니다. 다음 예제에서는 여러 가지 스타일을 한 문장 내에 적용하는 방법을 보여줍니다.

swift
import SwiftUI

struct ContentView: View {
    var body: some View {
        let fullString = "SwiftUI와 AttributedString을 사용하여 텍스트 스타일을 조정합니다."
        var attributedString = AttributedString(fullString)

        if let swiftUIRange = attributedString.range(of: "SwiftUI") {
            attributedString[swiftUIRange].font = .system(size: 20, weight: .bold)
            attributedString[swiftUIRange].foregroundColor = .blue
        }

        if let attributedRange = attributedString.range(of: "AttributedString") {
            attributedString[attributedRange].font = .italicSystemFont(ofSize: 20)
            attributedString[attributedRange].foregroundColor = .green
        }

        if let textStyleRange = attributedString.range(of: "텍스트 스타일") {
            attributedString[textStyleRange].underlineStyle = .single
            attributedString[textStyleRange].underlineColor = .purple
        }

        return Text(attributedString)
            .padding()
    }
}

struct ContentView_Previews: PreviewProvider {
    static var previews: some View {
        ContentView()
    }
}

이 예제에서는 다음과 같은 스타일을 적용했습니다:

  • "SwiftUI"는 굵고 파란색으로 설정
  • "AttributedString"은 기울임꼴과 초록색으로 설정
  • "텍스트 스타일"은 밑줄과 보라색으로 설정

이를 통해 AttributedString이 얼마나 유연하게 텍스트 스타일을 관리할 수 있는지 알 수 있습니다.

스타일 적용의 유용한 팁

1. 스타일을 적용할 범위 찾기

AttributedString에서 특정 단어 또는 문구의 범위를 찾기 위해 range(of:) 메서드를 사용합니다. 이 메서드는 해당 단어 또는 문구의 시작과 끝 인덱스를 반환하므로, 이를 통해 특정 부분에 스타일을 적용할 수 있습니다.

swift
if let range = attributedString.range(of: "SwiftUI") {
    attributedString[range].font = .system(size: 24, weight: .bold)
}

2. 여러 스타일을 동시에 적용하기

하나의 범위에 여러 스타일을 동시에 적용할 수 있습니다. 예를 들어, 글꼴과 색상을 동시에 변경할 수 있습니다.

swift
if let range = attributedString.range(of: "SwiftUI") {
    attributedString[range].font = .system(size: 24, weight: .bold)
    attributedString[range].foregroundColor = .blue
}

3. 동적으로 스타일 적용하기

동적 데이터에 따라 스타일을 적용할 때, forEach 루프를 사용하여 각 요소에 대해 스타일을 적용할 수 있습니다.

swift
let words = ["SwiftUI", "강력한", "프레임워크"]
words.forEach { word in
    if let range = attributedString.range(of: word) {
        attributedString[range].foregroundColor = .red
    }
}

실제 애플리케이션에서의 활용

실제 애플리케이션에서는 다음과 같은 상황에서 AttributedString을 활용할 수 있습니다:

  • 뉴스 앱: 기사 제목에서 특정 키워드를 강조하여 사용자가 주요 내용을 빠르게 파악할 수 있도록 함
  • 소셜 미디어 앱: 사용자 프로필에서 특정 단어(예: @사용자명, #해시태그)를 강조하여 가독성을 높임
  • 전자상거래 앱: 제품 설명에서 중요한 정보를 강조하여 고객의 관심을 끌도록 함

결론

Swift의 AttributedString을 사용하면 한 문장 내에서 특정 문자나 단어에 다양한 스타일을 적용할 수 있습니다. 이를 통해 텍스트의 가독성을 높이고, 사용자에게 중요한 정보를 쉽게 전달할 수 있습니다. 다양한 스타일 옵션을 활용하여 더욱 매력적이고 사용자 친화적인 인터페이스를 구축해보세요. SwiftUI와 AttributedString의 결합은 여러분의 앱을 한층 더 돋보이게 만들 것입니다.
