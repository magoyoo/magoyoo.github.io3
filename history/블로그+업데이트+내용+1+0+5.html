1.0.5 블로그 업데이트

1.0.5 업데이트 내용 📚

  • [CHANGED] Home 화면 폰트 사이즈 모바일 대응
  • [CHANGED] PostDetail 화면 폰트 사이즈 모바일 대응
  • [CHANGED] 블로그 컴포넌트 디자인 모바일 대응
  • [FEATURE] 모바일에서 홈 게시글 필터 적용
  • [FEATURE] Drawer 추가

모바일 화면 전용 Home을 새로 디자인 하였다. 모바일은 화면이 작기 때문에 다양한 컴포넌트가 혼재되어 있으면 지저분해 보인다. 특히 반응형 웹에서는 최대한 화면 구성을 간단히 하고 필터나 메뉴를 활용하여 사용자의 액션을 차별화 하는 것이 중요하다. 그래서 이번에 모바일 대응을하며 기존 "관심 받는 글"을 "인기글"이라는 필터로 따로 분류하여 재구성하였다. 또한 햄버거 메뉴를 추가하였다. 웹 화면에서 배너 형식으로 보여주던 블로그의 타이틀과 디스크립션을 메뉴에 넣고 검색 메뉴도 이동시켰다.

1.0.6 예정

  • About 페이지 신설
  • 세부적인 UI 개선

1.0.6 업데이트는 당분간 없을 것 같다. 지금 새로운 프로젝트를 계획하고 있고 조만간 생각이 정리될 것 같다. 평소에 있었으면 좋겠다 생각했던 서비스인데, 굳이 다른 사람들이 안 쓰더라도 내가 쓰는 것 자체가 의미가 있어 진행하기로 마음 먹었다. 프로젝트 가제는 "와쏘"이다. 빠르게 정리하고 빠르게 실행하자!
