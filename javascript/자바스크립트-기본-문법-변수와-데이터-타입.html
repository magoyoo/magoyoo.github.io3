자바스크립트 기본 문법: 변수와 데이터 타입

자바스크립트는 변수를 활용하여 데이터를 저장하고 다루는 강력한 기능을 제공합니다. 이번 글에서는 자바스크립트의 변수와 데이터 타입에 대해 쉽게 알아보겠습니다.

변수 선언과 할당

변수는 값을 저장하고 나중에 참조할 수 있는 식별자(identifier)입니다. 자바스크립트에서 변수를 선언하고 값을 할당하는 방법은 다음과 같습니다.

javascript
// 변수 선언과 값 할당
let myVariable = 10;

위 예제에서 let은 변수를 선언하는 키워드이고, myVariable은 변수 이름입니다. 등호(=)를 사용하여 변수에 값을 할당할 수 있습니다. 자바스크립트는 동적 타입 언어이기 때문에 변수의 타입은 자동으로 결정됩니다.

데이터 타입

자바스크립트는 여러 종류의 데이터 타입을 지원합니다. 대표적인 데이터 타입은 다음과 같습니다.

1. 숫자(Number)

javascript
let num = 123;
let decimal = 3.14;

2. 문자열(String)

javascript
let str = "Hello, JavaScript!";
let anotherStr = "Nice to meet you!";

3. 불리언(Boolean)

javascript
let isTrue = true;
let isFalse = false;

4. 배열(Array)

javascript
let fruits = ["apple", "banana", "orange"];

5. 객체(Object)

javascript
let person = {
  name: "Alice",
  age: 30,
  isAdmin: false,
};

6. undefined와 null

javascript
let undefinedVar = undefined;
let nullValue = null;

위와 같이 다양한 데이터 타입을 활용하여 자바스크립트에서 데이터를 다룰 수 있습니다.

결론

변수와 데이터 타입은 자바스크립트 프로그래밍의 기초이자 핵심입니다. 변수를 선언하고 데이터 타입을 이해함으로써 보다 효과적인 프로그래밍을 할 수 있습니다. 자바스크립트의 다양한 데이터 타입을 활용하여 웹 개발이나 앱 개발을 더욱 효율적으로 수행해보세요! 🚀
